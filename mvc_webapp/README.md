# mvc_webapp


### 项目说明
 - 一个应用了mvc设计模式的简单mvc案例
 
### 功能
 - 数据库查询、新增、删除、修改;
 - 模糊查询;
 - 删除提示、验证提示;
 - 验证用户名是否可用;


### 部署
